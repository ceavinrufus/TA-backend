## Getting started

```bash
# Clone the repository
git clone https://github.com/ceavinrufus/TA-backend.git

# Create environment variables file.
cp .env.example .env

# Install dependencies.
pnpm install
```

## Running the app

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```
